## 🚀 Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Anon23261/portfolio.git
   ```

2. **Navigate to Project**
   ```bash
   cd portfolio
   ```

3. **Open in Browser**
   ```bash
   # Using Python's built-in server
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
