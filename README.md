## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Git (for cloning)

### Quick Start
1. **Visit Live Site**
   ```bash
   https://anon23261.github.io/portfolio/
   ```

2. **Local Development**
   ```bash
   # Clone repository
   git clone https://github.com/Anon23261/portfolio.git
   
   # Navigate to project
   cd portfolio
   
   # Start local server (Python 3)
   python -m http.server 8000
   
   # Visit in browser
   http://localhost:8000
   ```
